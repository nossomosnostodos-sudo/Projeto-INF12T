# Projeto-INF12T
Este projeto é um sistema de gerenciamento de biblioteca, executado totalmente no terminal, que simula as operações básicas de um acervo real, como:

cadastro de livros

cadastro de autores

cadastro de usuários

empréstimos

reservas

listagem de registros

Apesar de ser simples, ele representa a lógica real usada em sistemas profissionais, servindo como base para evoluir para bancos de dados, interfaces gráficas e aplicações Web.

Conta com:

um sistema com SQLite

um sistema com MySQL / PostgreSQL

um sistema com interface gráfica (Tkinter, PyQt)

um sistema web (Django / Flask)
